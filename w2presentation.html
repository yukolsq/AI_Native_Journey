1. Simple Greeting (Variables)

This script is our very first step into Python! It introduces the concept of a "variable," which is like a container for storing information (in this case, a name). The script simply prints a pre-defined welcome message, showing how easy it is to display text on the screen.

Code: `welcome.py`

def print_welcome_message():
    name = "Yuko"
    message = f"Hello my name is {name}. I'm learning AI to become an awesome AI Native builder."
    print(message)

if __name__ == "__main__":
    print_welcome_message()

2. User Input (INPUT)

Building on the first script, this version introduces user interaction. We use the `input()` function to ask the user for their name. Whatever the user types in is then stored in a variable, and the script uses that input to create a personalized greeting, making the program more dynamic.

3. Making Decisions (If/Else)

This script demonstrates how programs can make decisions using `if/else` logic. It checks if the name the user entered matches a specific "special" name. If it matches, a unique greeting is displayed; otherwise, a general greeting is shown. This allows the program to respond differently based on conditions.

Code: `special_greeting.py`

# Define the special name you want to check for
# Remember to replace "[Your Name]" with the actual name you want to use, e.g., "Alice" or "John"
SPECIAL_NAME = "Yuko" 

# Ask the user for their name and store it in the 'name' variable
name = input("What's your name? ") 

# You can keep the 'goal' fixed or also make it ask for input
goal = "AI Native builder"

# Now, use an if/else statement to check the name
if name == SPECIAL_NAME:
    # Special welcome message for the designated name
    print(f"Hey, it's the awesome AI Director, {name}!")
else:
    # Regular greeting for any other name
    print(f"Hello, my name is {name}. I'm learning AI to become an awesome {goal}.")
